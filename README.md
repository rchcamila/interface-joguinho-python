# Joguinho em Python com Interface Gráfica

Este projeto é um joguinho interativo criado em Python para praticar lógica de programação, estruturas de controle e desenvolvimento de interfaces gráficas.

O usuário escolhe entre vários desafios matemáticos e lógicos, ganha pontos ao acertar e pode trocar de jogador.

## Conceitos praticados
- Estruturas condicionais (if / else)
- Laços de repetição (for e while)
- Funções e modularização
- Manipulação de strings
- Lógica matemática
- Interface gráfica com Tkinter
- Sistema de usuários e pontuação

## Funcionalidades
- Cadastro de jogador
- Sistema de pontuação
- Desafios:
  - Soma
  - Maior de 3 números
  - Par ou Ímpar
  - Fatorial
  - Tabuada
  - Soma de vários números
  - Média de 1 até N
  - Desafios aleatórios
- Troca de jogador
- Mensagem de despedida ao sair

## Como executar

1. Certifique-se de ter o Python instalado  
2. Baixe o projeto  
3. No terminal, execute.
